# CS-4365-Project

Checkpoint 2 Overview
This checkpoint of the project creates a TCP client/server network that allows clients to send messages to a server and receive an echo back of their message.
Installation
Clone the repository:
git clone https://github.com/Unwired8622/CS-4365-Project.git

python server.py
This starts the TCP server

In a separate terminal, run:
python client.py
This starts a TCP client
(Note that multiple clients can be created, but each one is created in its own separate terminal)
When prompted in the terminal, a client can send a message to be echoed back by the server.
Typing 'quit' will end the client's process.
